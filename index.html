El pretérito simple se usa para hablar de acciones terminadas en el pasado. Se usa para describir acciones puntuales y completas.

Explicación

El pretérito indefinido también se usa para hablar de acciones completadas en el pasado, pero se usa en contextos narrativos o para referirse a acciones repetidas.
